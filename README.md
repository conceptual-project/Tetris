# Тетрис / Tetris

Классический Тетрис с рендерингом WebGL2 и поддержкой геймпада.

## Возможности

- **Призрак фигуры** — полупрозрачная проекция внизу
- **Идеальная очистка** — бонус +2000×(уровень+1) за полную очистку поля
- **Таблица рекордов** — топ-10 с локальным хранением
- **Геймпад** — поддержка D-pad, стиков, кнопок (A/B/X/Y — вращение, LB/RB — сброс)
- **Двуязычность** — русский / English (переключение кнопкой)

## Управление

| Действие | Клавиатура | Геймпад |
|----------|-----------|---------|
| Движение | ← → | D-pad / Левый стик |
| Вращение | ↑ | A / B / X / Y |
| Мягкий сброс | ↓ | D-pad Down |
| Жёсткий сброс | Пробел | RB / LB |
| Пауза | P / Esc / Щелчок | Start |
